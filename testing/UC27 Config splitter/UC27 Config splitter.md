# UC27 Config splitter
## <i>configSplitter</i>

Configure an existent splitter by changing the values of its fields.

### Preconditions

### Steps
Change destination address and weight with the following restrictions:
 only the owner of the splitter can configure the primary destination, the owner of the secondary destination can change any of the
 5 managed entries.

### Postconditions
Returns true if assigning the provided values is successfully completed.
